Подготовка к уроку веб-разработки

Что взять с собой

  • Ноутбук с браузером Chrome или Firefox
  • Блокнот для конспекта
  • Флешка с резервной копией index.html

Порядок действий на уроке

  1. Создать файл index.html
  2. Вставить каркас с DOCTYPE
  3. Добавить заголовок и два абзаца
  4. Открыть страницу в браузере и сделать скриншот
