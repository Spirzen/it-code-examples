Частые вопросы

Нужен ли сервер, чтобы открыть HTML?

Нет. Достаточно сохранить файл и открыть в браузере. Сервер понадобится позже для форм и API.

Чем HTML отличается от CSS?

HTML задаёт структуру, CSS — внешний вид. Оба файла можно связать через <link rel="stylesheet">.
